# E-Commerce Sales Performance Analysis 

## About the Project
This project focuses on analyzing e-commerce sales data to understand sales trends, product performance, and city-wise profitability.

## Tools Used
- Microsoft Excel
- CSV Dataset

## Project Structure
- data/ : Raw dataset files  
- analysis/ : Excel-based analysis  
- visuals/ : Charts and visual outputs  

## Current Status
Initial project setup completed.  
Data analysis and visualizations will be added next.

## Objective
- Identify top-selling products  
- Analyze category-wise sales  
- Understand profit distribution across cities  

## Future Improvements
- Python-based analysis using Pandas  
- SQL queries for deeper insights  
- Interactive dashboards
